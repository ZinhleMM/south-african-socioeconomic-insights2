About This Project

This web application provides insights into South African households based on data from the General Household Survey (GHS) 2023. The application aims to make this valuable data more accessible and understandable through interactive visualizations and analysis.

Data Source

The data used in this application comes from Statistics South Africa's General Household Survey 2023. This annual survey collects data on:

  • Household demographics
  • Income and expenditure
  • Access to services
  • Food security
  • Living conditions
  • Asset ownership

Methodology

The application processes and analyzes data from 20,927 households across South Africa's nine provinces. The visualizations are created using:

  • Chart.js for interactive charts
  • MySQL database for data storage
  • Node.js backend for data processing

Key Features

  • Interactive data visualizations
  • Provincial comparisons
  • Household demographics analysis
  • Service delivery insights
  • Income distribution patterns

Citation
